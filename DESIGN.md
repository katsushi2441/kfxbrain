# Kurage FX Brain Design

- Kurage White Studio: ivory background, white panels, pale aqua and navy accents.
- Dense professional API workbench that fits at 1366x768 and remains usable on mobile.
- Body text 13-14px, controls 13px, headings 18-24px. The JSON editor and result are the visual focus.
- Never overlap controls and output. Long JSON must scroll inside its own panel.
